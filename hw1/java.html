All about Java

Java is a high level, object orriented language developed by James Gosling at Sun Microsystems in the 1990s. Java is a class based language meaning about everthing stems from a class. Whether it be the object class or a class you made. Java uses a garbage collection to automatically manage memory. Java is inspired or built off of C++. This means it has similar syntax and acts similar to C++ in ways like how types are declared, data containers like arrays and vectors(ArrayLists in Java), and classes or libraries used. Java is widely used in enterprise applications, Android mobile development, web servers, and even game development (like Minecraft).
